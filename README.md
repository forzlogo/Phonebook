<div align="center">
  <h1> «Умный» телефонный справочник </h1>
</div>

<br>

<div align="center">
  <h3>Описание</h3>
</div>
<p>Программа представляет собой телефонный справочник с возможностью управления контактами. Все данные сохраняются в файл и автоматически загружаются при запуске.</p>

<br>

<div align="center">
   <h3>Требования</h3>
  <p>Visual Studio с компилятором C++</p>
  <p>Операционная система Windows</p>
</div>

<br>

<div align="center">
  <h3>Возможности</h3>
</div>
<div>
  <p>Добавление и удаление контактов</p>
  <p>Редактирование контактов</p>
  <p>Поиск и сортировка по имени</p>
  <p>Сохранение и загрузка из файла</p>
</div>

<br>

<div align="center">
<h3>Запуск</h3>
</div>

<p>Создать проект</p>
<p>Добавить файлы main.cpp, contact.cpp, contact.h, phonebook.cpp, phonebook.h </p>
<p>Скомпилировать проект</p>
</div>
